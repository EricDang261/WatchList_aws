# WatchList_aws
The Watchlist is hosted by AWS S3 bucket. Backend is developed and deployed by Elastic Beanstalk.
These are services we were using for the application:
1. Elastic Beanstalk
2. DynamoDB
3. S3
4. MovieAPI
5. AnimeAPI
6. Exposing API for CRUD operations
